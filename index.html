How it works

Three steps.
Zero admin overhead.

01
Gemini Vision
AI reads the item

Upload a photo. Gemini extracts the object type, visible text (names, roll numbers, phone numbers), colour, brand, and condition — automatically.

Example: "Black leather wallet. Student ID: Arjun Sharma, MUJ 2021BCE0124."

02
Confidence Scoring
Every item gets scored

When a new submission arrives, AI compares it against all existing reports using category, colour, extracted text, brand, and location — each weighted by reliability.

Each comparison produces a 0–100% confidence score.

03
Smart Notification
Owner notified silently

If score ≥ 65%, the owner gets a direct notification with the finder's contact.

No public posts. No admin. No spam. Just a faster recovery — while both parties' privacy is protected until the match is confirmed.
